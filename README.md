### Home Rental Project ###

Holiday Cottages/Homes/Flats Rental Website

#### Functionalities ####

1. Users (signin, login, favorites, preferences, ...)
2. Multicriteria search : location, room number, price, ...
3. Admin
4. Stats

#### Technologies ####

+ Java/JEE - Spring
+ Hibernate
+ HTML (jsp, jstl tags), CSS, JS
